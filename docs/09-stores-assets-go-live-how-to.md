# Stores, Stock, Scaffold, Asset and Fleet Go-Live Guide

## Purpose

This guide explains how Stores Officers, Asset Officers, and operational initiators should use Southin PeoplePay / Operations Hub during the first controlled go-live phase.

The system currently supports:

- Microsoft 365 staff login
- Site and warehouse master data
- Site manager assignment
- Stores requisition capture
- Procurement request capture
- Approval inbox
- Imported stock opening balances
- Imported scaffold component register
- Imported asset and fleet register

## Important Go-Live Control

During the first go-live phase, the system should be used for request capture, approvals, and visibility.

Stock quantities must not be treated as automatically consumed until the stock issue, return, damaged, and quarantine workflows are fully tested and signed off.
