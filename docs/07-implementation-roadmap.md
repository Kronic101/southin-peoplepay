# Implementation Roadmap

## Phase 1: Foundation

- Create monorepo.
- Configure Next.js frontend.
- Configure NestJS backend.
- Connect Prisma to Supabase PostgreSQL.
- Create health endpoint.
- Create documentation structure.

## Phase 2: Authentication

- Microsoft 365 login foundation.
- Employee Number + PIN login.
- First login PIN change.
- Role-aware navigation.

## Phase 3: HR core

- Departments.
- Job titles.
- Sites.
- Employment types.
- Employee register.
- Employee portal account creation.

## Phase 4: Conditions of service

- Templates.
- Components.
- Employee assignment.
- Approved overrides.

## Phase 5: Attendance, leave, and casual labour

- Timesheets.
- Attendance approval.
- Overtime approval.
- Leave requests.
- Casual worker tracking.

## Phase 6: Payroll engine

- Payroll periods.
- Payroll runs.
- Earnings.
- Deductions.
- Statutory calculations.
- Approval workflow.
- Payroll lock.

## Phase 7: Reporting and SharePoint

- Executive dashboard API.
- SharePoint publishing.
- Statutory report archive.
- Audit report export.

## Phase 8: Pilot and go-live

- Import real employee data.
- Configure statutory rates.
- Run test payroll.
- Run parallel payroll.
- Validate against Zambian statutory requirements.
- Train HR, Finance, Directors, and Line Managers.
